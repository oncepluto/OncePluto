一、Oracle 11g Express Edition 介绍

Oracle Database 11g Express Edition是 Oracle 数据库的免费版本，支持标准版的大部分功能
做为免费的 Oracle 数据库版本，Express Edition的限制是：

  • 1）最大数据库大小为 11 GB
  • 2）可使用的最大内存是 1G
  • 3）一台机器上只能安装一个 XE 实例
  • 4）Express Edition只能使用单 CPU，无法在多CPU上进行分布处理
